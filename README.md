# AnagramFinder

**Задание:**

Написать метод, который принимает на вход 2 строки: s и p. Внутри себя метод должен находить все анаграммы p в строке s, и возвращать массив с начальными индексами этих анаграмм.

Строки состоят только из строчных английских букв, а длина строк s и p не превышает 20 100.

Порядок вывода не имеет значения.

## Использование

```ruby
anagram_finder = AnagramFinder::AnagramFinder.new("abc")
anagram_finder.all_indices_in("qabclbac") # => [1, 5]
```
